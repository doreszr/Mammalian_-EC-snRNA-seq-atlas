# Spatial navigation through evolution: a single-cell atlas of the mammalian entorhinal cortex

**DOI**

## Authors
Dorottya Maria Ralbovszki, Jesse J. Westfall, Yuki Mori, Stamatios A. Tahas, Mads F. Bertelsen, Irina Korshunova, Josefine Skov, Jan Gorodkin, Martin Hemberg, Stefan E. Seemann, Vanessa Hall, Menno P. Witter, Konstantin Khodosevich

## Brief description
Spatial navigation is a core cognitive function, relying on the integration of multisensory input and memory. The entorhinal cortex (EC), in close cooperation with the hippocampus, is central to this process and is remarkably conserved across mammals. However, how molecular signalling underlies both shared and species-specific navigational strategies remains unclear. Here, we present a cross-species single-cell atlas of the EC from human, baboon, mouse, and fruit bat - species spanning distinct evolutionary lineages and navigational demands, including true 3D navigation in bats. Using this resource, we identify conserved neuron populations as well as species-specific innovations, including fruit bat-specific superficial layer excitatory neuron subtypes. GABAergic neurons show strong conservation of SST and PV families, while VIP GABAergic neurons exhibit pronounced species-specific divergence, with an expanded repertoire in primates. In addition, whole-brain diffusion tensor imaging reveals conserved and species-specific connectivity between the MEC, hippocampus, and sensory cortices. These datasets provide valuable resources for investigating transcriptomics and connectivity that underlie spatial navigation across mammals.
