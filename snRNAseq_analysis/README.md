# snRNA-seq data analysis in Seurat

## 2_mouseAllen
Loading and subsetting EC reference data set from Allen mouse brain atlas.

## 3_Integration_annotation
Integrating samples by species and carrying out cross-species integration. Annotating species-specific and cross-species integrated data sets.
